# EjercicioPracticoSpringBoot
Ejercicio para reforzar SpringBoot. Consiste en crear dos tablas T_CONTACTS y T_PERSON juntando ambas obteniendo de resultado PersonaDto, controladores (vistas y API), bussiness (lógica de negocio), validaciones, dto...Test con JUNITs y MOCKITO con una cobertura del 90%, Colección POSTMAN y documentacion (CHANGELOD.md, README.md).
